What is JavaScript?

JavaScript enables users to use web pages interactively.  In order to benefit from all available web applications you will need to enable JavaScript in your web browser.


How to enable JavaScript in Netscape 4.x

To enable JavaScript using Netscape Navigator 4.x:

  1. Go to the "Edit" menu and select "Preferences". 
  2. Select the "Advanced" tab. 
  3. To enable JavaScript, select "Enable JavaScript".
  4. Click OK.

How to enable JavaScript in Internet Explorer 4.x

To enable JavaScript using Internet Explorer 4.x for Windows 95, 98 & NT:

  1. Go to the "View" menu and select "Internet Options."
  2. Select "Security" by clicking one on the word "Security." 
  3. Select the "Internet" Content Zone if it's not already selected.(Appears as an image of the Earth).
  4. Click the button marked "Custom".
  5. Click the "Settings" button.
  6. Scroll down to the section marked "Scripting".
  7. Under the "Active Scripting" section, select "Enable".
  8. Click OK.
To enable JavaScript using Internet Explorer 4.x for Macintosh:
  1. Go to the "Edit" menu and select "Preferences."
  2. Under the "Web Browser" section select "Web Content".
  3. Under the "Active Content" section select "Enable Scripting" so that a checkmark is displayed.
  4. Click OK.

How to enable JavaScript in Internet Explorer 5.x

To enable JavaScript using Internet Explorer 5.x for Windows 95, 98 & NT:

  1. Go to the "Tools" menu and select "Internet Options."
  2. Select "Security" by clicking one on the word "Security." 
  3. Select the "Internet" Content Zone if it's not already selected.(Appears as an image of the Earth)
  4. Click the button marked "Custom Level".
  5. Scroll down to the section marked "Scripting". 
  6. Under "Active Scripting" select "Enable."
  7. Click OK.

How to enable JavaScript for AOL

To enable JavaScript using AOL 4.0 for Windows 95 with Internet Explorer 4.x:

  1. In Windows 95 go to Start, Settings, Control Panel.
  2. Double-click the "Internet" icon.
  3. Select "Security" by clicking one on the word "Security." 
  4. Select the "Internet" Content Zone if it's not already selected.(Appears as an image of the Earth)
  5. Click the button marked "Custom Level".
  6. Scroll down to the section marked "Scripting of Java Applets" and select "Enable."
  7. Click OK.
To enable JavaScript using AOL 3.0 for Windows 95 with Internet Explorer 3.x:
  1. In Windows 95 go to Start, Settings, Control Panel.
  2. Double-click the "Internet" icon.
  3. Select "Security" by clicking one on the word "Security."
  4. Check the "Run ActiveX Scripts" box.
  5. Click OK.
