		Batch Files in the ChatRoom Application

Build.bat
	Builds the application.

startReg.bat
	Starts the rmi registry.

runSrv.bat
	Starts the server (it invokes startReg.bat)

client.bat
	Starts the client.  Format:
		start [username] [host] [message_file]

run.bat
	Test runs it by invoking runSrv and client.
